♻️ Projeto Integrador II – Plastec
📌 Sobre o Projeto

No segundo ano do curso, participei do Projeto Integrador II (Plastec), que teve como objetivo desenvolver uma solução sustentável para o reaproveitamento de resíduos plásticos.

O desafio principal foi transformar plástico triturado em filamentos para impressão 3D, criando um produto viável, acessível e alinhado aos princípios da economia circular.

🎯 Objetivos
Reutilizar resíduos plásticos como matéria-prima
Produzir filamentos para impressão 3D
Desenvolver uma solução de baixo custo
Aplicar conceitos de sustentabilidade e inovação
Avaliar a viabilidade prática e comercial do produto
⚙️ Desenvolvimento

Durante o projeto, o grupo realizou:

Estudo de formas de reaproveitamento do plástico
Definição do processo de produção dos filamentos
Testes de viabilidade do material para impressão 3D
Análise de custo e aplicabilidade no mercado
🌱 Sustentabilidade

O projeto foi baseado nos princípios da economia circular, buscando:

Redução de resíduos plásticos
Reutilização de materiais descartados
Diminuição do impacto ambiental
🧠 Aprendizados

A experiência proporcionou o desenvolvimento de diversas habilidades:

Planejamento de projetos
Trabalho em equipe
Resolução de problemas
Pensamento crítico
Aplicação prática de conhecimentos técnicos
Integração entre diferentes áreas (interdisciplinaridade)
🚀 Resultados

O projeto demonstrou que é possível transformar resíduos plásticos em um produto útil e sustentável, com potencial de aplicação real, especialmente em contextos educacionais e de baixo custo.

📎 Conclusão

O Plastec foi uma experiência importante para entender, na prática, como sustentabilidade, inovação e processos produtivos podem ser combinados para criar soluções eficientes e com impacto positivo.
